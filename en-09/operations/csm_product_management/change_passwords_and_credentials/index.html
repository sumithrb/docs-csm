Change Passwords and Credentials

This is an overarching procedure to change all credentials managed by Cray System Management (CSM) in HPE Cray EX system to new values.

There are many passwords and credentials used in different contexts to manage the system. These can be changed as needed. Their initial settings are documented, so it is recommended to change them during or soon after a CSM software installation.

Procedure

1. Change Hardware Credentials

  1. Perform procedures in Change Cray EX Cabinet Global Default Password.

  2. Perform procedures in Update Default Air-Cooled BMC and Leaf Switch SNMP Credentials.

  3. Perform procedures in Change Air-Cooled Node BMC Credentials.

  4. Perform procedures in Change SMNP Credentials on Leaf Switches.

  5. Perform procedures in Update Default ServerTech PDU Credentials used by the Redfish Translation Service (RTS).

  6. Perform procedures in Change Credentials on ServerTech PDUs.

2. Change Node Credentials

  1. Perform the procedure in in section 8.1.1 “Update NCN Passwords” in the HPE Cray EX System Administration Guide 1.4 S-8001.

  2. Perform the procedure in section 8.1.2 “Change Root Passwords for Compute Nodes” in the HPE Cray EX System Administration Guide 1.4 S-8001.

3. Change Service Credentials

  1. Perform the procedure in section 8.5.2 “Change the Keycloak Admin Password” in the HPE Cray EX System Administration Guide 1.4 S-8001.
